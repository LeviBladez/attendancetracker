# Attendance Tracker – Soroban Smart Contract

## Project Description

Attendance Tracker is a Soroban smart contract built on the Stellar blockchain that allows storing and managing student attendance records in a decentralized and tamper-proof way.
The contract ensures that attendance data cannot be modified once recorded, providing transparency and trust for educational institutions, training programs, and events.

This project demonstrates how Soroban smart contracts can be used for real-world record-keeping applications.

---

## What it does

* Stores attendance records on the Stellar blockchain
* Allows marking attendance for a student using their ID
* Prevents duplicate attendance for the same session
* Allows checking whether a student was present or absent
* Keeps records permanently on-chain

---

## Features

* ✅ Decentralized attendance storage
* ✅ Tamper-proof records
* ✅ Simple student ID based tracking
* ✅ Fast execution using Soroban
* ✅ Low transaction cost on Stellar
* ✅ Can be integrated with web or mobile apps
